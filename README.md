#  Heart Disease Prediction

A Machine Learning project for predicting heart disease using patient health data and the Random Forest algorithm.

## 📌 Overview

This project aims to predict whether a patient is at risk of heart disease based on medical attributes and clinical measurements. The dataset was cleaned, preprocessed, and used to train a Random Forest Classifier for accurate prediction.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## 🔍 Project Workflow

* Data Exploration and Analysis
* Data Cleaning
* Duplicate Removal
* Outlier Handling using IQR
* Feature Selection
* Train-Test Split
* Random Forest Model Training
* Model Evaluation

## 🤖 Model

Random Forest Classifier

```python
RandomForestClassifier(
    n_estimators=200,
    random_state=20
)
```

## 📊 Evaluation Metrics

* Accuracy Score
* Confusion Matrix
* Classification Report
